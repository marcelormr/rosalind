# Problem Summary

Confirm that Python is installed correctly by running:

`import this`

in the Python interpreter.

This prints the Zen of Python, a collection of design principles written by Tim Peters.

# What This Exercise Reinforced

- verifying a Python installation
- using the Python interpreter from the command line
- understanding basic module import syntax

# Takeaway

Although simple, this exercise establishes the working development environment
required for the rest of the Rosalind problems.

It also introduces Python’s emphasis on readability and simplicity through
the Zen of Python.
